# 自己写的小工具：清理新版QQ缓存

最近QQ for PC更新了，从老旧的客户端版本升级到最新的QQ nt测试版以后，以前自己写的清理QQ聊天缓存工具失效了，因此写了一个最新的，适用于最新版本QQ。

## 用法

- 清理新版QQ缓存.bat

下载它，放到任意地方，无需管理员权限，直接双击运行即可。



## 备注

- 纯Windows bat脚本实现，好处是Windows点击即用，坏处是仅限win平台。

- 源码看txt文件。

- 只删除聊天图片和视频，不影响文本聊天记录。
